RetroStore Features

RetroStore is open source and has the following features and goals:

  • Open API for retro clients to use
  • Open source code base
  • Platform for retro app and game developers to distribute their work

If you have an application, emulator or real hardware that you want to hook up to RetroStore, please contact us.
